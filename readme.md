# Automação das planilhas de Avaliação do CEAF - Versão 1.0
Primeira versão do descompilador de planilhas para SAFTEC/DASF/COAFE.
Funcionalidades:
- Seleção da planilha de avaliação
- Extratificação da planilha consolidada de avaliação por Unidade de Saúde
- Gerando assim, uma planilha para cada unidade e realizar encaminhamento para unidades.
